HTML is great to learn

There are a few reasons you should learn HTML:

  • websites are built with HTML
  • other libraries and languages use HTML, or HTML like markup languages
  • it's always great to learn new things
  • knowing HTML is a highly desirable skill to employers

Websites I like to visit:

  1. YouTube: it's a great place to learn a whole lot of different things for free.
  2. Stack Overflow: This is a great website for any coding questions you encounter.
  3. Duck Duck Go: Because I don't trust Google
